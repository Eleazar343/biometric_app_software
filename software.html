Nuevo Software: Protección Biométrica de Carpetas

Nuestro equipo ha desarrollado un innovador software para sistemas operativos Windows, que proporciona una nueva capa de seguridad para archivos y carpetas sensibles mediante el uso de reconocimiento facial en tiempo real.

🔐 ¿En qué consiste?

El software permite al usuario registrar su rostro mediante la cámara de la computadora. Una vez registrado, el programa vincula ese rostro a una o varias carpetas previamente seleccionadas. Solo el usuario autorizado podrá acceder a su contenido, garantizando que ningún otro tenga acceso, incluso si conoce la ubicación de la carpeta.

🧠 Funcionalidades destacadas

  • Registro facial con alta precisión usando inteligencia artificial.
  • Asociación de múltiples carpetas protegidas a un mismo usuario.
  • Posibilidad de añadir nuevos usuarios autorizados.
  • Interfaz intuitiva para seleccionar y gestionar carpetas protegidas.
  • Opciones para desactivar la protección o desvincular carpetas según necesidad.

📷 ¿Cómo funciona?

Cuando el usuario desea abrir una carpeta protegida, debe acceder a través del programa. Este activa la cámara, verifica la identidad del rostro registrado y, si coincide, permite abrir la carpeta. Si la verificación falla, el acceso es denegado.

🚀 Avances técnicos

  • Implementación con bibliotecas como OpenCV para reconocimiento facial.
  • Encriptado de rutas y nombres de carpetas para evitar accesos forzados.
  • Sistema de registros para intentos fallidos y alertas de seguridad.

💡 Ejemplo práctico

Imagina que trabajas con documentos legales confidenciales. Con este software, puedes vincular la carpeta "C:\Documentos\Legales" al rostro del abogado responsable. Solo él podrá abrirla, incluso si alguien más accede físicamente a la PC.

⚙️ Configuración flexible

El software permite que cada usuario personalice el nivel de seguridad, cambie de cámara, configure recordatorios de actualización facial y administre accesos de terceros.
